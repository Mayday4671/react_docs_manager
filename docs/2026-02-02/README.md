## 🔧 技术栈

### 前端
- **框架**: Next.js 15 (App Router)
- **UI库**: Ant Design 5.x
- **样式**: CSS Modules + Tailwind CSS
- **状态管理**: React Hooks
- **类型检查**: TypeScript

### 后端
- **运行时**: Node.js
- **API**: Next.js API Routes
- **数据库**: SQLite (开发) / PostgreSQL (生产)
- **ORM**: Prisma

### 开发工具
- **包管理**: npm
- **代码规范**: ESLint + Prettier
- **版本控制**: Git

## 🚀 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 初始化数据库
```bash
npx prisma generate
npx prisma db push
npx prisma db seed
```

### 3. 启动开发服务器
```bash
npm run dev
```

### 4. 访问系统
- 地址: http://localhost:3001
- 管理员: admin / admin123
- 普通用户: user / user123

## 📝 开发规范

### 文件命名
- 组件文件: PascalCase (如 `UserManagement.tsx`)
- 工具文件: camelCase (如 `userService.ts`)
- 样式文件: kebab-case (如 `user-management.css`)

### 目录结构
```
src/
├── app/                    # Next.js App Router
│   ├── api/               # API 路由
│   ├── layout.tsx         # 根布局
│   └── page.tsx           # 首页
├── frontend/              # 前端代码
│   ├── components/        # React 组件
│   ├── styles/           # 样式文件
│   └── utils/            # 工具函数
└── backend/               # 后端代码
    ├── services/         # 业务逻辑
    ├── models/           # 数据模型
    └── utils/            # 工具函数
```

### 代码规范
- 使用 TypeScript 进行类型检查
- 遵循 ESLint 规则
- 组件使用函数式组件 + Hooks
- API 使用 RESTful 设计
