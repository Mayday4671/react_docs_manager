# 海康API文档界面优化 - 树形分类+抽屉详情

## 优化概述

根据用户需求，重新设计了海康威视API文档的界面布局，采用左右分栏的设计：
- **左侧**：树形结构展示API分类
- **右侧**：列表展示选中分类下的API接口
- **详情**：使用抽屉方式展示API详细信息

## 界面设计改进

### 1. 整体布局 (左右分栏)

```
┌─────────────────────────────────────────────────────────┐
│                    统计面板 (4个卡片)                      │
├─────────────────┬───────────────────────────────────────┤
│   左侧分类树     │           右侧API列表                  │
│                │                                       │
│ 📁 设备管理 (2)  │  GET  获取设备列表                     │
│   ├─ API1      │  POST 获取设备详情                     │
│   └─ API2      │                                       │
│ 📁 视频管理 (2)  │  [详情] [调用] 按钮                    │
│ 📁 用户管理 (2)  │                                       │
│ 📁 事件管理 (1)  │  分页控制                             │
└─────────────────┴───────────────────────────────────────┘
```

### 2. 左侧分类树特性

- **树形结构**：清晰的层级关系
- **图标标识**：文件夹图标 + API图标
- **数量徽章**：显示每个分类下的API数量
- **方法标签**：不同颜色区分HTTP方法
- **状态标记**：弃用API的红色标签
- **搜索功能**：支持关键词搜索，结果替换树形显示

### 3. 右侧API列表特性

- **列表展示**：简洁的API信息展示
- **方法头像**：HTTP方法作为列表项头像
- **状态标签**：弃用、需认证等状态标记
- **操作按钮**：详情查看、模拟调用
- **调用统计**：显示API调用次数
- **版本信息**：API版本标签
- **分页控制**：支持分页浏览

### 4. 抽屉详情面板

- **宽屏设计**：720px宽度，充分展示内容
- **分区展示**：基本信息、接口说明、请求参数、响应信息、备注说明
- **JSON格式化**：美化的JSON代码展示
- **操作便捷**：顶部快速调用按钮

## 交互流程优化

### 1. 浏览流程
1. 用户进入页面，查看统计概览
2. 左侧树形结构选择感兴趣的分类
3. 右侧列表展示该分类下的所有API
4. 点击"详情"按钮查看完整API文档

### 2. 搜索流程
1. 在左侧搜索框输入关键词
2. 搜索结果替换树形显示
3. 可以直接从搜索结果查看API详情

### 3. 调用流程
1. 在列表或详情页点击"调用"按钮
2. 系统记录调用次数
3. 实时更新统计信息
