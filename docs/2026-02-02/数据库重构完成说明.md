## 📊 数据库表结构详情

### sys_user (系统用户表)
```
id          - 主键
username    - 用户名（唯一）
password    - 密码
email       - 邮箱（唯一）
phone       - 手机号
avatar      - 头像URL
status      - 状态（1:启用 0:禁用）
roleId      - 角色ID
createdBy   - 创建人ID
updatedBy   - 更新人ID
createdAt   - 创建时间
updatedAt   - 更新时间
```

### sys_role (系统角色表)
```
id          - 主键
roleName    - 角色名称（唯一）
roleKey     - 角色标识（唯一）
description - 角色描述
status      - 状态
createdBy   - 创建人ID
updatedBy   - 更新人ID
createdAt   - 创建时间
updatedAt   - 更新时间
```

### sys_menu (系统菜单表)
```
id          - 主键
key         - 菜单标识（唯一）
label       - 菜单名称
icon        - 图标
path        - 路由路径
component   - 组件路径
parentId    - 父菜单ID
orderNum    - 排序号
menuType    - 菜单类型（M:目录 C:菜单 F:按钮）
visible     - 是否可见（1:显示 0:隐藏）
status      - 状态（1:启用 0:禁用）
createdBy   - 创建人ID
updatedBy   - 更新人ID
createdAt   - 创建时间
updatedAt   - 更新时间
```

### sys_log (系统日志表)
```
id          - 主键
module      - 模块名称
action      - 操作类型
method      - 请求方法
url         - 请求URL
ip          - IP地址
userAgent   - 用户代理
params      - 请求参数
result      - 操作结果
errorMsg    - 错误信息
costTime    - 耗时(ms)
userId      - 操作用户ID
username    - 操作用户名
status      - 状态（1:成功 0:失败）
createdAt   - 创建时间
```

### tbl_notification (通知公告表)
```
id          - 主键
title       - 标题
content     - 内容
type        - 类型（info, success, warning, error）
priority    - 优先级（0:普通 1:重要 2:紧急）
status      - 状态（1:已发布 0:草稿）
readCount   - 阅读次数
publishAt   - 发布时间
expireAt    - 过期时间
createdBy   - 创建人ID
updatedBy   - 更新人ID
createdAt   - 创建时间
updatedAt   - 更新时间
```

## 🔌 API接口文档

### 用户管理

#### 获取用户列表
```
GET /api/users?page=1&pageSize=10&status=1

Response:
{
  "data": [...],
  "total": 100,
  "page": 1,
  "pageSize": 10,
  "totalPages": 10
}
```

#### 创建用户
```
POST /api/users
Content-Type: application/json

{
  "username": "test",
  "password": "test123",
  "email": "test@example.com",
  "phone": "13800138000",
  "roleId": 1
}
```

### 通知管理

#### 获取通知列表
```
GET /api/notifications?page=1&pageSize=10&type=info&status=1

Response:
{
  "data": [...],
  "total": 50,
  "page": 1,
  "pageSize": 10,
  "totalPages": 5
}
```

#### 创建通知
```
POST /api/notifications
Content-Type: application/json

{
  "title": "系统通知",
  "content": "通知内容",
  "type": "info",
  "priority": 0,
  "status": 1
}
```

```
Response:

  "pageSize": 20,

}
```

## 📝 使用示例

### 1. 查询用户列表
```bash
curl http://localhost:3001/api/users
```

### 2. 创建新用户
```bash
curl -X POST http://localhost:3001/api/users \
  -H "Content-Type: application/json" \
  -d '{
    "username": "newuser",
    "password": "password123",
    "email": "newuser@example.com"
  }'
```

### 3. 查询通知
```bash
curl http://localhost:3001/api/notifications?page=1&pageSize=5
```

### 4. 查询日志
```bash
curl http://localhost:3001/api/logs?module=user&action=CREATE
```
