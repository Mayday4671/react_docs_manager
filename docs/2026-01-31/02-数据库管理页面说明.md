# 数据库管理页面 - 总览所有表

## 🎯 新增功能

创建了一个**数据库管理总览页面**，可以：
- 📊 查看所有数据库表的统计信息
- 🚀 快速访问各个管理页面
- 📈 实时显示每张表的记录数
- 🔍 按类型筛选（系统表/业务表）

---

## 📋 页面位置

**菜单路径**: 首页 → 数据库管理  
**访问路径**: `/database`  
**图标**: DatabaseOutlined 🗄️

---

## ✨ 功能特性

### 1. 统计卡片
显示 4 个关键指标：
- **数据库表总数**: 8 张表
- **系统表**: 5 张（sys_*）
- **业务表**: 3 张（tbl_*）
- **总记录数**: 所有表的记录总和

### 2. 快速访问卡片
以卡片形式展示所有表：
- 每张表显示图标、名称、记录数
- 点击卡片直接跳转到对应管理页面
- 不同颜色区分不同表

### 3. 详细表格
表格形式展示所有表的详细信息：
- **表名**: 数据库表名（如 sys_user）
- **显示名称**: 中文名称（如 用户管理）
- **类型**: 系统表/业务表
- **记录数**: 当前表的记录总数
- **描述**: 表的功能说明
- **操作**: 管理按钮，跳转到管理页面

### 4. 分类标签页
- **全部表**: 显示所有 8 张表
- **系统表**: 只显示 sys_* 表
- **业务表**: 只显示 tbl_* 表

### 5. 实时刷新
点击刷新按钮，重新获取最新的统计数据

## 🔧 技术实现

### 前端组件
**文件**: `src/frontend/components/system/DatabaseManagement.tsx`

**主要功能**:
- 使用 Ant Design 组件（Card, Table, Statistic, Tabs）
- 使用 Next.js Router 进行页面跳转
- 实时获取统计数据
- 响应式布局

### API 端点
**文件**: `src/app/api/database/stats/route.ts`

**功能**:
- 并发查询所有表的记录数
- 返回表的详细信息（名称、描述、路由等）
- 分类标记（系统表/业务表）

**返回数据格式**:
```json
{
  "success": true,
  "data": [
    {
      "name": "sys_user",
      "displayName": "用户管理",
      "icon": "UserOutlined",
      "color": "#1890ff",
      "count": 2,
      "description": "系统用户信息...",
      "route": "/system/users",
      "category": "system"
    },
    ...
  ]
}
```
