# 系统名称动态显示功能

## 🎯 功能说明

现在系统名称会从数据库的系统配置中读取，并动态显示在以下位置：
1. **页面顶部** - 显示系统名称（如 "My App"）
2. **浏览器标签页** - 显示 "页面名称 - 系统名称"（如 "系统配置 - Next.js Admin"）

---

## ✨ 实现功能

### 1. 从数据库读取系统名称
- 读取 `sys_config` 表中 `configKey = 'system.name'` 的配置
- 默认值：`My App`
- 自动刷新：每 30 秒检查一次配置更新

### 2. 动态显示位置

#### 页面顶部（Header）
```
┌─────────────────────────────────┐
│ 🎨 [系统名称]  菜单1  菜单2  菜单3 │
└─────────────────────────────────┘
```

#### 浏览器标签页
```
[页面名称] - [系统名称]
例如：
- 首页 - Next.js Admin
- 系统配置 - Next.js Admin
- 用户管理 - Next.js Admin
```

## 🔧 技术实现

### 状态管理
```typescript
const [systemName, setSystemName] = useState('My App');
```

### 获取系统配置
```typescript
useEffect(() => {
  const fetchSystemConfig = async () => {
    const response = await fetch('/api/configs');
    const data = await response.json();
    const systemNameConfig = data.data?.find(
      (config: any) => config.configKey === 'system.name'
    );
    if (systemNameConfig) {
      setSystemName(systemNameConfig.configValue);
    }
  };

  fetchSystemConfig();
  
  // 每30秒刷新一次
  const interval = setInterval(fetchSystemConfig, 30000);
  return () => clearInterval(interval);
}, []);
```

### 更新浏览器标题
```typescript
// 方式1：菜单点击时更新
document.title = `${label} - ${systemName}`;

// 方式2：标签页切换时更新
useEffect(() => {
  const currentTab = items.find(item => item.key === activeKey);
  if (currentTab) {
    const title = typeof currentTab.label === 'string' 
      ? currentTab.label 
      : '页面';
    document.title = `${title} - ${systemName}`;
  }
}, [activeKey, items, systemName]);
```

## 📝 使用方法

### 1. 修改系统名称

1. 访问 **系统管理 → 系统配置**
2. 找到 `system.name` 配置项
3. 点击 **编辑**
4. 修改配置值（如改为 "我的管理系统"）
5. 点击 **确定** 保存

### 2. 查看效果

修改后，系统会在以下时机更新显示：
- **立即生效**：刷新页面
- **自动更新**：等待 30 秒（自动刷新配置）

## 🔄 自动刷新机制

### 刷新频率
- **间隔时间**：30 秒
- **触发条件**：组件挂载后自动开始
- **清理机制**：组件卸载时自动停止

### 为什么需要自动刷新？
1. 多用户环境下，其他用户可能修改了系统配置
2. 无需手动刷新页面即可看到最新配置
3. 提升用户体验

### 如何立即看到效果？
如果不想等待 30 秒，可以：
1. 刷新浏览器页面（F5）
2. 切换到其他页面再切换回来

## 📊 配置项说明

### system.name
- **配置键**：`system.name`
- **配置类型**：系统配置
- **默认值**：`Next.js Admin System`
- **用途**：显示在页面顶部和浏览器标签页
- **建议长度**：4-20 个字符

### 其他相关配置
- `system.version`：系统版本号
- 可以根据需要添加更多配置项

## ⚠️ 注意事项

### 1. 配置键不可修改
编辑配置时，`configKey` 字段是禁用的，只能修改 `configValue`

### 2. 刷新延迟
自动刷新有 30 秒延迟，如需立即看到效果请手动刷新页面

### 3. 默认值
如果数据库中没有 `system.name` 配置，会显示默认值 `My App`

### 4. 特殊字符
系统名称支持中英文、数字、空格等常用字符
