# 首页布局紧凑化与视觉修正计划 (V4)

针对 V3 版本过于拥挤导致出现滚动条的问题，V4 将重心放在“精简”与“一屏化”上。

## 拟議变更

### 1. 结构精简 (Structural Simplification)

- **从 2x2 回退到 1+2 布局**:
  - 上方保持“全系统访问趋势”面积图（宽度 2/3）。
  - 右侧叠放“设备在线状态”环形图（高度减半）。
  - 下方新增一排极简的“资源监控”微型组件（或直接移除多余图表），确保不超出底边界。
- **一屏显示 (One-Screen View)**: 严格控制所有 Card 和图表的高度，确保在标准化屏幕（1080p）下无需滚动。

### 2. 高度微调 (Height Micro-adjustment)

- 面积图高度从 420px 降至 300px。
- 其他辅助图表高度降至 180px - 200px。
- 移除底部的“系统公告”卡片，释放垂直空间。

### 3. 边框保留与优化

- 保留 V3 增强的主题色边框，但略微调薄（从 1.5px 降至 1px），减少视觉压迫。

## 验证计划

### 自动化验证

- 检查 `@ant-design/plots` 组件是否渲染正常。

### 手动验证

1. **浏览器预览**: 确认首页在不缩放的情况下不再出现垂直滚动条。
2. **布局平衡**: 检查图表是否均匀分布在屏幕内，没有大面积留白。
3. **视觉层次**: 确认卡片边框在紧凑布局下依然清晰且美观。
