# 首页高级美化与布局优化实现计划 (V3)

用户反馈首页卡片边框不明显且图表区域显得空旷。本计划将通过增加图表密度和强化视觉层次来解决。

## 拟議变更

### 1. 布局扩充 (Layout Expansion)

- **图表矩阵化**: 将目前的 1 (面积图) + 1 (环形图) 布局升级为 **2 x 2 矩阵布局**，增加两个小微图表。
  - **新增：资源负载 (Radar/Liquid Chart)**: 展示 CPU/内存实时水位。
  - **新增：存储状态 (Progress Bar/Radial Chart)**: 监控磁盘空间占用。
- **动态高度**: 将图表容器高度从 350px 适当增加至 400px+，填补视觉空降。

### 2. 边框与阴影效果增强 (Visual Borders)

- **卡片样式升级**:
  - 弃用系统默认的微弱边框，改用更有质感的 `1.5px solid` 主题色半透明边框。
  - 增加常态下的微弱阴影 `box-shadow`，悬浮时进一步强化，使卡片产生“悬浮感”。
  - 适配暗色模式。

### 3. 数据 Mock 与组件更新

- 引入 `Radar` 和 `TinyArea` (或其他 AntV 精简组件)。
- 统一所有卡片的 `borderRadius` 为 16px，提升现代化感。

## 验证计划

### 自动化验证

- 检查 `@ant-design/plots` 导出组件是否包含所需的 Radar 或 Liquid。

### 手动验证

1. **视觉检查**: 观察四个卡片入口是否具有清晰的边框和层次分明的阴影。
2. **布局检查**: 确认首页形成了平衡的图表矩阵，下方不再有大面积空白。
3. **交互检查**: 所有的卡片点击和悬浮效果是否依然丝滑。
