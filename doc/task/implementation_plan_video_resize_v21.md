# 视频播放器自适应修复计划 (V21)

解决视频播放器在侧边栏收缩、窗口缩放或标签页切换后无法自动填充新空间的问题。

## 拟議变更

### 1. 播放器容器感应增强 (H5Player.tsx)

- **引入 ResizeObserver**: 监听播放器父级 `div` 的尺寸。
- **动态 Resize 调用**: 当容器宽高发生变化（即便 `window` 没变）时，显式调用播放器实例的 `jsplayer.JS_Resize()` 方法。

### 2. 针对 Keep-Alive 的显示补偿

- **可见性监听**: 在 `H5Player.tsx` 中通过 `useEffect` 监听组件挂载与容器变化。
- **展示瞬间延迟重绘**: 当组件从隐藏变为可见时，执行 `JS_Resize()`。

### 3. 代码健壮性

- 确保在组件销毁时正确关闭 `ResizeObserver`，防止内存泄漏。

## 验证计划

1. **侧边栏测试**: 在视频播放页面，反复点击收缩/展开左侧菜单，观察视频画面是否实时填充右侧空间。
2. **标签切换测试**: 切到首页再切回视频页，确保画面比例依然正常且无多余拉伸。
3. **窗口缩放测试**: 手动拉伸浏览器，验证画面自适应速度。
