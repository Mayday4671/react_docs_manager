# 首页底部卡片样式精修与 Alert 化 (V7)

彻底消除滚动条并引入更直观的 `Alert` 风格列表，打造完美的固定视口大屏。

## 拟議变更

### 1. 结构压实 (Height Compression)

- **图表降高**: 访问趋势和设备图表高度从 `320px` 降至 `280px`。
- **快捷入口间距**: 进一步减少顶层入口和下方组件的 `margin`。
- **精确定高**: 底部卡片内容区固定为 `210px`，确保 1080p 分辨率下绝无滚动条。

### 2. Alert 风格化集成

- **通知公告**:
  - 弃用简单列表，采用 `Alert` 组件或自定义 `div` 实现带色块背景、带左侧粗线条、带语义化图标的消息项。
  - 不同类型（Error, Warning, Info, Success）对应不同色系，与用户截图保持一致。
- **更新日志**:
  - 同样采用轻量级 Alert 样式，突出版本号和日期，增加功能标签背景色。

### 3. 组件列表优化

- 每组数据严格限制展示 4 条，防止内容撑破容器。
- 增加整体容器的 `overflow: hidden`。

## 验证计划

1. **绝对无滚动条**: 在标准 16:9 分辨率下验证右侧滚动条是否消失。
2. **Alert 视觉对标**: 确认背景色块、图标布局是否符合用户上传的截图审美。
3. **响应式锁定**: 检查在小屏下是否依然能保持紧凑而不溢出。
