# 首页全响应式自适应改造 (V12)

彻底摆脱固定像素（Hard-coded px）限制，实现基于 Flexbox、相对单位和容器自适应的高级 Dashboard 布局。

## 拟議变更

### 1. 高度占位解耦

- **全局容器**: 强制外层容器 `height: 100%` 或 `calc(100vh - 头部/页脚高度)`。
- **占比分配**:
  - 快捷入口: `flex-shrink: 0` (固定内容高度)。
  - 中部图表: 改为 `flex: 2` (相对于底部卡片占据更多空间，而非固定 280px)。
  - 底部卡片: 改为 `flex: 1.5`。

### 2. 元素自适应增强

- **图标容器**: 将快捷入口的图标容器由固定 `38px` 改为相对单位（如 `min(5vw, 42px)`）或使用媒体查询调整。
- **字体缩放**: 引入全局 `clamp` 机制，使标题和内容在不同屏幕宽度下自动调整比例。
- **图表适配**: 设置 `@ant-design/plots` 的 `autoFit: true`，确保图表随容器宽高变化实时重绘。

### 3. 排版逻辑重构

- 统一样式变量: 使用 Ant Design 的 `token` 动态生成间距，避免手动填入 `16px`。
- 响应式栅格: 优化 `Row/Col` 的 `xs`, `sm`, `md`, `lg` 断点配置，确保窄屏下不发生重叠。

## 验证计划

1. **多屏幕模拟**: 在浏览器 DevTools 中测试从 1366px 到 3840px (4K) 的全范围适配。
2. **内容溢出检查**: 确认图标和文字在极窄屏幕或极扁屏幕下依然清晰、不遮挡。
3. **性能稳定性**: 验证高度频繁变化（ResizeObserver 触发）时，布局是否稳定。
