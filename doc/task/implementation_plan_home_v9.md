# 首页底部卡片压力测试与滚动优化 (V9)

在用户自定义高度的基础上，通过扩充数据量和引入内部平滑滚动，确保信息展示的完整性与交互体验。

## 拟議变更

### 1. 数据量扩充

- **通知公告**: 手动增加 10 条 Mock 数据，类型涵盖 `new`, `warning`, `info`。
- **更新日志**: 增加 10 条版本历史，模拟真实的业务演进。

### 2. 卡片内部滚动优化

- **滚动条开启**: 将底部卡片内容区的 `overflow: hidden` 改为 `overflow-y: auto`。
- **视觉修正**: 消除由于高度不匹配导致的“内容显示一半”现象。
- **样式美化**: 引入自定义 CSS 优化内部滚动条（变细、圆角、主题色），使其在 Windows 系统下也具备极简美感。

### 3. 全局布局稳固

- 确保内部滚动不会触发 `Layout` 层级的全局滚动条，维持首屏全览的底线。

## 验证计划

1. **压力测试**: 确认 10+ 条数据下，卡片内部滚动是否顺畅。
2. **边缘检查**: 检查列表底部是否有足够的 `padding`，防止最后一条数据显示不全。
3. **样式适配**: 切换主题色，观察滚动条颜色是否同步。
