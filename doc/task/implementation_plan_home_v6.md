# 首页底部卡片视觉增强计划 (V6)

在保持全屏适配的基础上，进一步提升底部信息展示的丰富度与专业感。

## 拟議变更

### 1. 结构与尺寸优化

- **高度调增**: 将通知公告和更新日志卡片的内容区高度由 `160px` 调整至 `220px` - `240px`。
- **动态适配**: 精确控制顶部快捷入口和中部图表的高度，腾出空间给底部，确保整体不触发垂直滚动条。

### 2. 消息内容精细化

- **图标化展示**:
  - **通知公告**: 根据消息类型（重要、提醒、新）配以不同的 Ant Design 图标（如 `NotificationFilled`, `InfoCircleFilled`, `ExclamationCircleFilled`）。
  - **更新日志**: 使用带有图标的列表项或 `Timeline` 组件，增加版本更新的仪式感。
- **增强样式**: 对每一条消息采用微型容器封装，增加背景色悬浮反馈。

### 3. 组件替换

- 对于重要的通知消息，考虑使用 `Alert` 组件的精简样式或自定义带图标的横条。

## 验证计划

1. **一屏全览**: 检查各主流分辨率下页面底边界是否溢出。
2. **图标表现**: 确认各类型消息对应的图标语义准确且颜色协调。
3. **视觉空隙**: 确认底部卡片加高后，页面下方的空白区域已被合理解构。
