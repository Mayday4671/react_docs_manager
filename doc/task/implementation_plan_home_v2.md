# 首页功能升级与图表美化实现计划

用户反馈首页图表美观度不足，且“视频预览”无法直接跳转。本计划将解决这些痛点。

## 拟議变更

### 1. 导航跳转支持 (Navigation Support)

- **全局跳转机制**:
  - 为了让 `Home.tsx` 能触发 `LayOut.tsx` 中的菜单切换，我们将创建一个简单的自定义事件监听机制，避免复杂的 Context 重构。
  - 在 `LayOut.tsx` 挂载时监听 `switchTab` 事件。
- **快捷入口联动**:
  - 点击“视频预览”卡片，自动触发事件，让 Layout 切换到 `h5-player` 标签页（如果已打开则激活，未打开则新建）。

### 2. 图表美化与增强 (Chart Enhancement)

- **柱状图 -> 平滑面积图 (Area Chart)**:
  - 增加 `smooth: true` 让曲线更柔和。
  - 使用 `areaStyle` 添加主题色渐变，增加视觉层次感。
  - 优化 X 轴和 Y 轴网格线。
- **饼图 -> 环形图 (Donut Chart)**:
  - 增加 `innerRadius: 0.6` 转换为环形，并在中心显示“总计”文本。
  - 启用标注联动。
- **Mock 数据扩充**:
  - 提供更真实的 24 小时或 7 天数据波动。

### 3. 布局微调

- 为卡片增加图标背景的毛玻璃或微光效果（可选）。
- 确保所有颜色通过 `token` 动态注入。

## 验证计划

### 自动化验证

- 运行 `npm run dev` 检查控制台是否有 AntV 渲染报错。

### 手动验证

1. **跳转测试**: 点击首页第一个快捷入口“视频预览”，检查是否成功跳转到 `H5-Player` 页面并自动激活 Tab。
2. **样式检查**:
   - 确认柱状图是否变为了漂亮的渐变面积图。
   - 确认饼图是否变为了环形图，且鼠标悬停有反馈。
3. **响应式**: 缩放浏览器窗口，确认图表自适应容器大小。
